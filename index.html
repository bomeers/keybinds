Useful Keybinds
1

ctrl + T

Open a new tab

2

ctrl + W

Close selected tab

3

Middle Mouse Button or the "scroll wheel button"

Close clicked tab

4

ctrl + click

To select tabs

4a

ctrl + click and ctrl + R

To refresh multiple selected tabs

4b

ctrl + click and ctrl + W

To close multiple selected tabs

5

shift + click

To select tabs in a row (select start and end tabs)

6

ctrl + tab and ctrl + shift + tab

To navigate tabs to the right with ctrl + tab, or to the left with ctrl + shift + tab

7

ctrl + shift + K

To duplicate tabs

8

ctrl + shift + W

Close all tabs

9

ctrl + shift + T

Restore previously session

9a

ctrl + shift + T

This also restores previously closed tabs

10

Grouping tabs

In Microsoft Edge, you can group tabs together by dragging and dropping

11

ctrl + shift + esc

Open the Windows Task Manager

12

ctrl + c and windows + V

Use windows clipboard to copy multiple items with ctrl + c and windows + V (windows = start menu key)

13

windows + ⬆️⬇️⬅️➡️(arrow keys)

Move selected windows around workspace with the windows and arrow keys

14

windows + tab

Open windows workspaces or create new workspaces

15

ctrl + windows + ⬆️⬇️⬅️➡️

Quickly switch between windows workspaces

16

windows + X

Open windows quick tool menu

17

windows + 1-9

Open pinned taskbar item
